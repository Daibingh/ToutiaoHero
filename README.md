# 头条英雄答题助手

项目依赖

- python 3.6 以及相关包

  ```txt
  jieba==0.39
  opencv_python==4.1.0.25
  requests==2.18.4
  wxpy==0.3.9.8
  selenium==3.141.0
  pywin32==222
  numpy==1.16.1
  baidu_aip==2.2.18.0
  Pillow==6.2.1
  PyHook3==1.6.1
  ```

- adb 调试工具

- Chrome 浏览器和对应的 chrome driver

- 手机或模拟器 (注：模拟器只能观战无法答题；若使用真机数据通过 usb 线传输稍有耗时)
